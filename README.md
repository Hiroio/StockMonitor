# FinanceMonitor

FinanceMonitor is an iOS app for tracking stock market data in real-time.  
It uses the **financialdata.net API** to fetch stock information and visualize it with interactive charts.

---

## Features

1. **Stock List**  
   - Browse a list of available stocks.  
   - Data is retrieved directly from the financialdata.net API.

2. **Stock Details & Charts**  
   - Tap on a stock to see a detailed view with a **candlestick chart** showing Open, High, Low, Close (OHLC) and trading volume.  
   - Charts automatically adapt to screen size and scale dynamically for better readability.

---

## Technologies

- SwiftUI  
- Swift Charts  
- Git & GitHub  
- [financialdata.net API](https://financialdata.net)
